# VaultMind

VaultMind 是一个独立桌面 App，用于管理个人知识库、远程知识源、代码项目和本地安全保险箱。

## 功能

- 本地邮箱/用户名/密码注册登录
- 本地 SQLite 数据库存储用户、同步记录、下载后的内容
- 飞书 OAuth 登录
- 选择本地文件并加密上传，或直接录入文本密钥并加密上传
- 使用 AES-256-GCM 加密，PBKDF2-SHA256 派生密钥
- 加密口令只在本机输入，不会上传到飞书
- 上传位置默认是飞书云空间根目录，也可以填写文件夹 token
- 下载飞书加密文件并解密入库
- 文本内容默认用星号遮挡，输入本地密码后解锁查看
- 文件内容显示原路径，可输入本地密码后保存回本地

## 飞书开放平台配置

创建飞书应用后，在应用安全设置里添加重定向 URL：

```text
http://127.0.0.1:37891/feishu/oauth/callback
```

应用需要开通云空间文件上传/下载相关权限，以及用户身份读取权限。App 内会显示建议 scope。

## 运行

```bash
npm install
npm start
```

## 注意

- 当前直传限制为单个本地文件约 18MB，避免超过飞书直传限制。
- 飞书加密口令忘记后无法恢复已上传文件内容。
- 本地登录密码用于保护 SQLite 里的下载内容，忘记后无法解锁本地内容。
- SQLite 文件保存在 Electron 的 userData 目录，App 内可点击“打开 SQLite 位置”。
